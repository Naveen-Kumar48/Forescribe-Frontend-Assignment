# ✨ Modern Testimonial Grid

A premium, interactive masonry grid showcasing company partnerships and user testimonials. Built with high-performance animations and a sleek dark-themed aesthetic.

## 🚀 Features

- **Dynamic Masonry Layout**: Smart grid system that scales from mobile to high-resolution displays (up to 1500px).
- **Interactive Animations**: Smooth entrance animations and hover effects powered by `motion/react` (Framer Motion).
- **Premium Design Logic**: Alternating vertical offsets for a deep, professional look.
- **Responsive Components**: Tailored experiences for mobile, tablet, and desktop views seamlessly.
- **Modern Styling**: Curated color palettes and glassmorphism using Tailwind CSS.

## 🛠️ Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Typeface**: [Inter](https://fonts.google.com/specimen/Inter)

## 🏁 Getting Started

### Prerequisites

- **Node.js**: 18.x or later
- **npm**: 9.x or later

### Installation

1. **Clone the repository**
   ```bash
   # If you haven't cloned yet
   cd testimonial-grid
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **View the result**
   Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📁 Key File Locations

- **Main Logic**: `app/page.tsx` (Contains grid data and structure)
- **Global Config**: `app/layout.tsx` (SEO and Fonts)
- **Custom Components**: `components/`
- **Global Styles**: `app/globals.css`
